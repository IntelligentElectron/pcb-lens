# Tools

| Tool | Description |
|------|-------------|
| [`get_design_overview`](get_design_overview.md) | Get metadata, layer stackup, component/net counts, and section sizes for an IPC-2581 file |
| [`query_components`](query_components.md) | Find components by refdes pattern with placement coordinates and BOM data |
| [`query_net`](query_net.md) | Query a net by name pattern for pin connections, routing per layer, and via info |
| [`query_constraints`](query_constraints.md) | Query layout constraints from a Cadence `.tcfx` file |
| [`export_cadence_board`](export_cadence_board.md) | Export a Cadence Allegro `.brd` file to IPC-2581 XML (Windows only) |
| [`export_cadence_constraints`](export_cadence_constraints.md) | Export a Cadence Allegro `.brd` file to `.tcfx` constraint XML (Windows only) |
